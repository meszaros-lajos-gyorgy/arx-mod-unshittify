--------------------------------
 Unshittify mod for Arx Fatalis
--------------------------------

A mod that aims to increase clarity of puzzles and in-game mechanisms

--------------
 Installation
--------------

Unpack the contents of this zip and add it into
your Arx Fatalis/Arx Libertatis game's root folder

-----------
 Uninstall
-----------

Remove the folders you've copied from src in Arx Fatalis/Arx Libertatis
game root
